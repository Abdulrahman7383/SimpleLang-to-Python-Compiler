# SimpleLang-to-Python-Compiler
اسم المشروع: SimpleLang-to-Python-Compiler
=========================================

مرحباً بك في مستودع مشروع SimpleLang-to-Python-Compiler.

وصف المشروع:
-----------
هذا المشروع عبارة عن مترجم (Compiler) مصمم لترجمة لغة برمجة مبسطة وتحويلها إلى كود بايثون (Python) قابل للتشغيل. يهدف المشروع إلى تطبيق المفاهيم الأساسية لبناء المترجمات وفهم مراحل معالجة الشفرة المصدرية.

مراحل عمل المترجم:
-----------------
1. التحليل اللغوي (Lexical Analysis): 
   يقوم المترجم بقراءة الكود المصدري للغة المبسطة وتقسيمه إلى وحدات ورموز أساسية (Tokens) يسهل التعامل معها.

2. التحليل النحوي (Syntax Analysis): 
   يتم فحص الرموز (Tokens) للتأكد من مطابقتها للقواعد النحوية للغة المبسطة، وبناء هيكل برمجي سليم.

3. توليد الكود (Code Generation): 
   في هذه المرحلة النهائية، يتم تحويل الهيكل النحوي السليم إلى كود بايثون (Python) مكافئ وجاهز للتنفيذ.

كيفية التشغيل:
-------------
1. قم بتنزيل ملفات المشروع على جهازك.
2. استخدم بيئة التشغيل المناسبة لتشغيل المترجم (حسب لغة البرمجة التي استخدمتها في بناء المترجم نفسه).
3. قم بتزويد البرنامج بملف يحتوي على كود باللغة المبسطة.
4. سيقوم البرنامج بتوليد ملف بايثون (.py) يحتوي على الكود المترجم.
